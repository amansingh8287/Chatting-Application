# 💬 Real-Time Chatting Application

A full-stack real-time chat application built using the MERN stack with Socket.IO for instant messaging.

---

## 🚀 Features

* 🔐 User Authentication (Signup / Login)
* 💬 Real-time messaging (Socket.IO)
* 🟢 Online / Offline user status
* 👤 Avatar support
* ⚡ Instant message delivery
* 📱 Responsive UI (mobile friendly)

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Redux Toolkit
* Tailwind CSS

### Backend

* Node.js
* Express.js
* Socket.IO

### Database

* MongoDB Atlas

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/amansingh8287/Real-Time-Chatting-Application.git
cd Real-Time-Chatting-Application
```

---

### 2. Install Dependencies

#### Backend

```
cd backkend
npm install
```

#### Frontend

```
cd frontend
npm install
```

---

### 3. Setup Environment Variables

Create a `.env` file inside the backend folder and add:

```
MONGO_URI=your_mongodb_uri
JWT_SECRET_KEY=your_secret_key
```

---

### 4. Run the Application

#### Backend

```
cd backkend
npm start
```

#### Frontend

```
cd frontend
npm start
```
